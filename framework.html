Frameworks

Un framework es una estructura de soporte utilizada como base para el desarrollo de software.

Ventajas

  • Facilitan el desarrollo rápido
  • Están bien documentados
  • Promueven buenas prácticas

Desventajas

  • Limitan la personalización
  • Pueden ser complejos de aprender

Frameworks Frontend más utilizados

  • React
  • Angular
  • Vue.js

Frameworks Backend más utilizados

  • Laravel
  • Django
  • Express.js
