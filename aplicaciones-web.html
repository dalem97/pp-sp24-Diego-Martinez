Desarrollo Web

El desarrollo web se divide en dos partes principales:

  • Frontend: Se encarga de la interfaz de usuario. Tecnologías usadas:
    • HTML
    • CSS
    • JavaScript
  • Backend: Se encarga de la lógica del servidor. Tecnologías más comunes:
    • Bases de Datos
    • Servidor Web
    • Lenguajes de Programación: PHP, Python, Node.js, etc.
